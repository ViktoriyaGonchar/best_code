# Best Code - Free Code Library

## Project Description

**Best Code** is a free library of ready-made, tested code snippets for developers. Copy and paste solutions in 7 programming languages: JavaScript, Python, Java, C++, PHP, Ruby, Go.

![Banner](images\best_code.png)

## Features

### 1. Best Code 
Main library with tested code instructions for all 7 programming languages. Each instruction includes:
- Task name (3-7 words)
- Description (where it works and who needs it)
- Ready-to-use working code
- Explanations for beginners

### 2. Share Code Here! 
Public library for code sharing:
- Publish snippets up to 500 characters
- Sidebar filter for sorting by programming languages
- Unlimited number of articles (no restrictions)
- Automatic sorting by language and title in alphabetical order
- Voting system: ✅ (Works) and ❌ (Doesn't work)
- View all articles or filter by specific language

**Filtering:**
- "All Languages" button — shows all articles
- Language buttons (JavaScript, Python, Java, C++, PHP, Ruby, Go) — filter articles by selected language
- Articles are sorted first by language (alphabetically), then by title (alphabetically)

### 3. Documentation 
List of free and authoritative documentation sources:
- Read the Docs
- GitHub Docs
- MDN Web Docs
- Python.org Docs

Security policy and code warnings.

- Orange color palette in Viking style

- Smooth transitions and hover effects
  

## Voting System

Each article has:
- **✅ Works**: number of functionality confirmations 
- **❌ Doesn't work**: number of problem reports 

## Installation and Launch

1. Download project files
2. Open `index.html` in a browser
3. Start using the library!

No server installation or additional dependencies required.
