# Best Code - Бесплатная библиотека кода

## Описание проекта

**Best Code** — это бесплатная библиотека готовых, проверенных фрагментов кода для разработчиков. Копируйте и вставляйте решения на 7 языках программирования: JavaScript, Python, Java, C++, PHP, Ruby, Go.

## Возможности

### 1. Best Code 
Основная библиотека с проверенными инструкциями по коду для всех 7 языков программирования. Каждая инструкция включает:
- Название задачи (3-7 слов)
- Описание (где работает и кому нужно)
- Готовый рабочий код
- Пояснения для новичков

### 2. Поделись кодом тут! 
Публичная библиотека для обмена кодом:
- Публикация фрагментов до 500 символов
- Боковой фильтр для сортировки по языкам программирования
- Бесконечное количество статей (без ограничений)
- Автоматическая сортировка по языку и названию в алфавитном порядке
- Система голосования: ✅ (Работает) и ❌ (Не работает)
- Просмотр всех статей или фильтрация по конкретному языку

**Фильтрация:**
- Кнопка "Все языки" — показывает все статьи
- Кнопки для каждого языка (JavaScript, Python, Java, C++, PHP, Ruby, Go) — фильтруют статьи по выбранному языку
- Статьи сортируются сначала по языку (алфавитно), затем по названию (алфавитно)

### 3. Документация 
Список бесплатных и авторитетных источников документации:
- Read the Docs
- GitHub Docs
- MDN Web Docs
- Python.org Docs

Политика безопасности и предупреждения о коде.

## Технологии

- HTML5
- CSS3 (градиенты, анимации, адаптивность)
- JavaScript (Vanilla JS, без фреймворков)
- LocalStorage для хранения данных

## Языки интерфейса

- Английский (по умолчанию)
- Русский

Переключение языков доступно в правом верхнем углу страницы.

## Особенности дизайна

- Оранжевая цветовая палитра в стиле викинга
- Градиенты и анимации
- Адаптивный дизайн для мобильных устройств
- Плавные переходы и hover-эффекты


## Система голосования

Каждая статья имеет:
- **✅ Работает**: количество подтверждений работоспособности 
- **❌ Не работает**: количество сообщений о проблемах 

## Установка и запуск

1. Скачайте файлы проекта
2. Откройте `index.html` в браузере
3. Начните использовать библиотеку!

Не требует установки сервера или дополнительных зависимостей.

## Структура файлов

- `index.html` — основной файл приложения
- `index0.html` — пример заполненного аккаунта пользователя Ulylanja
- `README_RU.md` — документация на русском языке
- `README_EN.md` — документация на английском языке
- `Инструкция.md` — инструкция по обнулению контейнеров и аккаунтов

## Ограничения

- Нет изображений, аудио и видео файлов
- Нет внешних CDN (кроме согласованных)
- Нет вредоносного JavaScript
- Нет доступа к localStorage, cookies, geolocation, camera
- Нет ссылок на сторонние вредоносные сайты
